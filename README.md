siff2013
========

2013 Seattle International Film Festival Film Scheduler

Tool to automatically build a schedule of films showing at the 2013 Seattle International
film festival.  You can use the tool at: http://www.tonydicola.com/siff2013/

Created by Tony DiCola (tony@tonydicola.com)

Thanks to help from:
- AngularJS
- Bootstrap
- Underscore.js
- Mocha, Chai
- Yeoman
